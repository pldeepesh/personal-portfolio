# KPI Definitions

## Organic traffic
- **Organic impressions**: Search impressions from Google Search Console.
- **Organic clicks**: Search clicks from Google Search Console.
- **Average position**: Weighted average position for target cluster queries.

## Indexation health
- **Indexed URLs**: Count of valid indexed pages for `/blog/` and `/topics/`.
- **Coverage exceptions**: Excluded URLs requiring remediation.

## Conversion
- **CTA conversion rate**: `cta_click / organic_sessions`.
- **Waitlist conversion rate**: `waitlist_submit / products_page_sessions`.
- **Newsletter conversion rate**: `newsletter_submit / blog_sessions`.

## Engagement
- **Deep-read rate**: `post_read_75 / post_sessions`.
- **Topic click-through**: `topic_click / blog_index_sessions`.
